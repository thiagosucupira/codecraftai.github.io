@Test
public void testUserAuthentication_ValidCredentials_ReturnsUser() {
    // Arrange
    String email = "user@example.com";
    String password = "validPassword";
    User expectedUser = new User(1L, email);
    when(userRepository.findByEmail(email)).thenReturn(expectedUser);
    when(passwordEncoder.matches(password, expectedUser.getPassword())).thenReturn(true);
    
    // Act
    User result = userService.authenticate(email, password);
    
    // Assert
    assertNotNull(result);
    assertEquals(expectedUser.getId(), result.getId());
    verify(userRepository).findByEmail(email);
}

@Test
public void testUserAuthentication_InvalidCredentials_ThrowsException() {
    // Arrange
    String email = "user@example.com";
    String password = "invalidPassword";
    User user = new User(1L, email);
    when(userRepository.findByEmail(email)).thenReturn(user);
    when(passwordEncoder.matches(password, user.getPassword())).thenReturn(false);
    
    // Act & Assert
    assertThrows(AuthenticationException.class, () -> {
        userService.authenticate(email, password);
    });
}

Operação Autônoma

O CodeCraftAI opera de forma autônoma em um container Docker na nuvem da empresa, realizando as seguintes ações sem intervenção direta:

🔄

Monitoramento Contínuo

Monitora repositórios Git em busca de novos commits e alterações que possam afetar a cobertura de testes.

📊

Análises Agendadas

Executa análises completas de repositórios em horários pré-definidos (ex: diariamente às 3h da manhã).

📝

Geração Proativa de Testes

Identifica classes com cobertura insuficiente e implementa testes unitários automaticamente.

📨

Comunicação Automática

Envia relatórios, alertas e notificações por email e Teams com base em eventos ou agendamentos.

🔄

Integração com CI/CD

Responde a eventos de pipeline CI/CD, analisando resultados de build e testes.

Tecnologias e Frameworks

O CodeCraftAI utiliza um conjunto robusto de tecnologias modernas para oferecer automação inteligente de sugestões de PR e implementação de testes unitários, integrando-se perfeitamente com suas ferramentas existentes.

Inteligência Artificial
LLMs (Large Language Models)
Modelos de linguagem avançados como GPT-4 para geração de código e compreensão de contexto.
RAG (Retrieval-Augmented Generation)
Sistema que combina recuperação de informações com geração de texto para análise precisa de documentação.
Backend
Python
Linguagem principal para o backend, processamento de dados e integração com LLMs.
LangChain
Framework para integração de LLMs com outras ferramentas e fontes de dados.
Bases de Dados Vetoriais
FAISS ou Pinecone para armazenamento eficiente de embeddings de documentos.
Integrações
GitPython
Biblioteca para operações Git como criação de branches, commits e PRs.
APIs REST
Integração com Jira e Confluence para acesso a tickets e documentação.
JUnit
Framework para testes unitários em Java, garantindo cobertura de código.
Microsoft Graph API
Para envio de emails e integração com Microsoft 365.
Microsoft Teams API
Para envio de notificações e interação via chat no Teams.
Infraestrutura
Docker
Containerização para implantação consistente e isolada.
Cloud Deployment
Implantação em nuvem (AWS/Azure/GCP) para escalabilidade e processamento de grandes volumes de documentação.
Webhooks
Para integração em tempo real com sistemas Git, Jira e Confluence.
Kubernetes
Para orquestração de containers em ambiente de produção.

Compatibilidade e Extensibilidade

O CodeCraftAI foi projetado para ser altamente compatível com ambientes de desenvolvimento existentes e facilmente extensível para novas ferramentas e linguagens. A arquitetura modular permite adicionar suporte para novas linguagens de programação além do Java, bem como integrar com sistemas adicionais conforme necessário.
